Character Generation Twitter Bot

For my final project of Creative Programming, I decided to create a character generation bot. Inspired by the Emoji Mashup Bot, this bot would post hourly with a new character creation featuring a visual representation of the character, randomly generated name, class, gender, and more. While the bot is no longer live, artists, designers, and other creatives can use this bot as character inspiration for games, DnD, overcoming art block, etc. You can see an example post here and here.



Development Process

Art

Before I programmed a single line of code, I sat down and designed all of the body options as well as some of the hair and accessories. To do this, I used an app called Procreate and sketched the basic body dimensions I wanted. I had to ensure each component would look cohesive and line up properly, so I designed each item on its own layer from the bottom up, starting with the skin and finishing with accessories. By designing first in layers, I could make sure everything overlapped in the right spots so I knew which order to draw the images in the code. Once I was finished, I exported each layer as a transparent PNG.


Code

This bot was made using Processing, an open-source Java graphics library built for artists and visual designers. Using this library I was able to create an application that could generate characters at the press of a button, and post them to my Twitter account with another press, complete with animations and post confirmations. My favorite part about this project is how easy it is to add more hair/skin/body/etc options- just drop the new assets in the right folder and the code will automatically detect and them to the components array the next time it runs.
