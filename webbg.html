Website Background

When I redesigned my website in late 2019/early 2020, I knew I wanted to have a better, more visually engaging landing page. Previously, I had used a screenshot of a piece of art I made with Processing for my Creative Programming class. I wanted to keep a similar design, so I decided to animate that original design.



Code

The website background was made using Processing, an open-source Java graphics library built for artists and visual designers. Before the animation, the program simply picked a random location, drew a dot, drew a line in a random direction in a certain range of length, and then repeated for some number of times. To animate this, I had to rethink how I was drawing the connections. Now, on every frame, each dot looks for new dots within a certain threshold. If both dots are not currently at "max connection capacity", a line will be drawn between them. If the dots move too far apart, the line disconnects. The dots move randomly across the canvas and bounce off the edges. To export as a gif, I recorded my screen for about 30 seconds, used Vegas Pro to make the video loop, and then used an online converter to turn the new video into a gif. This gif is used as the background of my homepage, and a static screenshot is used on every other page.
