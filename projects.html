Game Development Projects

----- FORCE -----

April 2018 - Game Jam B - Partner Project

Created in one week and placing 3rd in SGDC's Game Jam B, FORCE is a 2-4 player arena-style fighting game in which you can only kill your opponents via the environment. To play, you will need at least one friend, at least two Xbox controllers, and a Windows PC. Each character has unique abilities, and there are many ways to die, including by your own hand. Play some rounds in the arena- the first player to 5 or 9 points will win the match. Created with help from Adam Gincel.

----- cope -----

September 2018 - Game Jam C - Solo Project

A solo project made for SGDC's 48 hour Game Jam C. Placed 5th overall and 1st in Visual Design. cope is a single player platformer featuring world duality. It requires a Windows machine and a keyboard to play. Press space to jump, arrow keys to move left and right, and shift to see secret platforms as you navigate your brain through scary and dangerous places and back into your skull. To close the game, press escape. A speedrun of the game jam build can be found below.




----- FORCE 2 -----

November 2018 - Game Jam D - Group Project

After accidentally deleting all the source code for FORCE, FORCE 2 was rebuilt from the ground up for SGDC's week-long Game Jam D, and placed 3rd overall. Adam came back from FORCE to help re-write some code, and Lachlan Wright was brought on to help with music and character design, as well as debugging.
